Rule 1: Affirmative (+) Sentences

Hi! I'm Terry the Turtle!

The Present Simple is for habits and facts!

  • For I, you, we, they, use the base verb. (e.g., I play)
  • For he, she, it, add -s or -es to the verb. (e.g., He plays)

Exercise 1: Let's Practice!

2. They (watch) movies on Fridays.
3. The cat (sleep) on the sofa.
4. We (eat) breakfast at 8:00 AM.
5. She (study) hard for her exams.

Rule 2: Negative (-) Sentences

Hello! I'm Penny the Penguin!

To make a negative sentence, we use helpers!

  • Use don't (do not) for I, you, we, they. (e.g., You don't play)
  • Use doesn't (does not) for he, she, it. (e.g., She doesn't play)
  • The main verb stays in its base form! No -s!

Exercise 2: Make it Negative!

3. We (not go) to school on Sunday.
4. My mom (not work) on weekends.
5. They (not speak) French.

Rule 3: Questions (?)

Hey! I'm Fiona the Fox!

To ask a question, start with a helper!

  • Start with Do for I, you, we, they. (e.g., Do you play?)
  • Start with Does for he, she, it. (e.g., Does he play?)
  • Again, the main verb is always in its base form!
