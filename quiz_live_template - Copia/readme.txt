Come creare un altro file per quiz live per una classe

Devi cambiare in template 5ce con il nome della classe nuova
Cambia col nome della classe anche nella variabile console di create_alunni.py
Devi mettere i nomi degli alunni
Devi mettere i nomi degli alunni anche in crea_link_alunni poi esegui il codice, copia quello che viene mostrato e salvalo nel file accesso_alunni.html

A questo punto agli alunni dai accesso_alunni.html
tu invece apri copia_in_console.html
